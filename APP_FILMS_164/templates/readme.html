📌 Présentation

Ce projet est une application web de gestion complète pour un salon de coiffure / barbershop, développée avec Python Flask.

Il s'agit d'un projet personnel pour mes frères, réalisé dans le cadre du module 164.

🔧 Technologies utilisées

  • Python 3 + Flask
  • HTML / CSS / Bootstrap (si utilisé)
  • SQLite ou MySQL (selon ta base)
  • WTForms
  • Jinja2 (template Flask)

🧠 Fonctionnalités principales (CRUD)

  • Employés : création, édition, suppression, spécialités, disponibilité
  • Clients : gestion des infos personnelles et préférences
  • Services : types de prestations (coupe, barbe, soins...)
  • Rendez-vous : prise de rdv, historique, statut

🗃️ Base de données

L’application gère plusieurs tables relationnelles :

  • clients
  • employes
  • services
  • rencontrer
  • evaluer
  • executer

🚀 Lancer l’application

Utilise le fichier run_mon_app.py :

python run_mon_app.py

Par défaut, l’application est disponible sur http://127.0.0.1:5000
